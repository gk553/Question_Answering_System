# 📚 Question Answering System using NLP & Transformers

## 📌 Project Description
This project is a Question Answering (QA) System that answers user questions based on a given document using Natural Language Processing and Transformer models.

The system reads a document and extracts accurate answers using a pretrained NLP model.

---

## 🚀 Features
- Ask questions from custom documents
- Transformer-based NLP model
- Fast Gradio Web Interface
- Lightweight and easy to run
- Beginner-friendly AI project

---

## 🧠 Tech Stack
- Python
- NLP
- Hugging Face Transformers
- Gradio
- PyTorch

## ⚙️ Installation

### 1. Clone Repository
```
git clone <your-repo-link>
cd qa-system
```

### 2. Install Dependencies
```
pip install -r requirements.txt
```

---

## ▶️ Run Application
```
python app.py
```

Application runs at:
```
http://127.0.0.1:7860
```
